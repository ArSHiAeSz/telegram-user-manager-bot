# Telegram Registration Bot (Python)

A simple Telegram bot built with Python that allows users to register using the `/register` command.
User data is stored locally using SQLite.

---

## Features
- Telegram bot using python-telegram-bot
- User registration command
- Prevents duplicate registrations
- SQLite database for persistent storage
- Clean and modular code structure

---

## Technologies Used
- Python 3
- python-telegram-bot
- SQLite (built-in)
